Research : LDRails

LDRails

LDRails is an open-source project that demonstrates how to extend a Grails scaffold to produce Linked Data with minimal effort by equating entities in the domain model with real world objects (RWOs). The principles involved could be applied to any MVC framework that supports scaffolds.

The following URIs illustrate the major categories of entities in a domain model:

Model
  • Real World Model URI
  • Model Web Document URI (HTML)
  • Model Web Document URI (RDF)
Class
  • Real World Class URI
  • Class Web Document URI (HTML)
  • Class Web Document URI (RDF)
Instance
  • Real World Instance URI
  • Instance Web Document URI (HTML)
  • Instance Web Document URI (RDF)
Attribute
  • Real World Attribute URI
Relationship
  • Real World Relationship URI

Note that the default content-negotiation behavior for Real World Object URIs is to return HTML. This is to encourage the use of RWO URIs in place of Web Document URIs. The "format=rdf" is not part of the RWO URI, but it is being used here to force the Grails content-negotiation mechanism to return RDF for demonstration purposes.

Also beware that I do not recommend these particular URI patterns. They were inherited from Grails.
